What is Cloud Computing?

Cloud Computing is the delivery of computing resources such as servers, storage, networking, databases, analytics and software over the Internet. Instead of buying physical hardware, users rent resources from cloud providers like Microsoft Azure, Amazon Web Services (AWS), and Google Cloud Platform (GCP).

Learn Azure Hosting

Major Cloud Services

IaaS

Infrastructure as a Service provides virtual machines, storage, networking and servers. Example: Azure Virtual Machines.

PaaS

Platform as a Service provides an environment to develop and deploy applications. Example: Azure App Service.

SaaS

Software as a Service delivers complete software through the web. Example: Microsoft 365 and Gmail.

Advantages of Cloud Computing

Cost Effective

Pay only for the resources you use.

Scalable

Increase or decrease resources whenever needed.

Secure

Cloud providers offer encryption, backups and identity management.

Global Access

Access applications and data from anywhere.

How to Host a Static Website on Microsoft Azure

  1. Sign in to Azure Portal.
  2. Create a Storage Account.
  3. Open the Storage Account.
  4. Select Static Website under Data Management.
  5. Enable Static Website Hosting.
  6. Upload index.html, CSS, JavaScript and image files.
  7. Save the configuration.
  8. Copy the generated URL.
  9. Open the URL in your browser to view the website.
